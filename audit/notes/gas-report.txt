CONFIGURATION
=============
The project's foundry.toml includes `gas_reports = ["*"]` which enables
gas reporting for all contracts when tests are present.

Gas reports will be automatically generated once test functions are written.
Run `forge test --gas-report` or `forge snapshot` after adding tests to
produce per-function gas cost estimates.

GAS SNAPSHOT
============
No gas snapshot was generated because no test functions exist yet.
The source code was reconstructed from Etherscan/Sourcify, which does
not include tests.

To generate:
  1. Write test contracts (e.g., in audit/tests/ or test/)
  2. Run: forge test --gas-report
  3. Or:  forge snapshot --snap audit/notes/gas-snapshot.txt
